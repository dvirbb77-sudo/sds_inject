PROJECT OVERVIEW
================================================================================

A production-grade, enterprise-ready Kubernetes bootstrap automation framework
that produces a single self-executing installer artifact capable of deploying
complete Kubernetes environments on clean Ubuntu 22.04 systems.

Key Features:
  ✓ Self-contained artifact (single .run file)
  ✓ Offline installation capability
  ✓ Master and worker node support
  ✓ Automatic state detection
  ✓ Comprehensive error handling
  ✓ Structured logging framework
  ✓ Pre-flight validation
  ✓ CI/CD integration (Jenkins)
  ✓ VM testing (Vagrant)
  ✓ Remote deployment utilities
  ✓ Complete documentation

================================================================================
DEPLOYMENT SCENARIOS
================================================================================

Scenario 1: Single-Node Development
  1. Run: k8s-installer.run --master
  2. Result: Single control-plane node (master + worker)

Scenario 2: Multi-Node Cluster
  1. Master: k8s-installer.run --master
  2. Get token: kubeadm token create
  3. Workers: k8s-installer.run --worker --master-ip <ip> --join-token <token>
  4. Result: 1 master + N worker nodes

Scenario 3: Automated Deployment
  1. build.sh → creates dist/k8s-installer.run
  2. scp installer to target
  3. cd/deploy.sh → remote execution
  4. Result: Automated multi-system deployment

Scenario 4: CI/CD Pipeline
  1. Push code to GitHub
  2. Webhook triggers Jenkins
  3. Pipeline runs all stages
  4. Artifact archived
  5. Result: Automated build, test, and artifact generation
